# Wendigo Band Website

Official website for Wendigo, an alternative rock band. Features music releases, upcoming shows, and band information.

## Features

- Band biography and photos
- Music releases showcase (albums, EPs, singles)
- Social media integration (Spotify, Instagram, YouTube, TikTok, Bandcamp, Bandsintown)
- Live performance statistics
- Contact information
- Responsive design with modern UI
- Built with React 19 and TypeScript
- Styled with CSS and Helvetica Neue typography

## Tech Stack

- **Frontend**: React 19, TypeScript
- **Build Tool**: Vite
- **Styling**: CSS with custom components
- **Icons**: Custom SVG icons

## Development

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

```bash
npm install
```

### Development Server

```bash
npm run dev
```

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Project Structure

src/
